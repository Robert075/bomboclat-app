## Project Description
A web application to find recipes by ingredients and automatically calculate their complete macronutrient profile. This app integrates TheMealDB and CalorieNinjas APIs to simplify the process of carbohydrate and calorie counting, making it easier for users to manage specific dietary plans, such as those for diabetes.

## Target Audience
Individuals dealing with diabetes who are looking for a simple, fast and reliable tool to count carbohidrates and calories.
